# Home

This plugin inspired by [dataview](https://github.com/blacksmithgu/obsidian-dataview) plugin for Obsidian. It allows you to query your markdown files and render the results as tables or lists. Eventhough syntax is similar to dataview plugin, it's more SQL like.

## Installation

Install plugin via pip:

```bash
pip install mkdocs-dataview-plugin
```

Add plugin to mkdocs.yml:

```yaml
plugins:
  - dataview
```

Check [book library](examples/library/index.md) for usage examples.
